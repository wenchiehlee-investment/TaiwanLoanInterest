# TaiwanLoanInterest

台灣貸款利率監控與資料收集系統

## 專案目標

建立一個自動化系統，監控並收集台灣各類貸款利率資料，提供：
- 原始數據收集（直接從央行、各銀行官網抓取）
- 歷史資料庫（長期累積，供研究分析）
- 開放資料格式（JSON/CSV 匯出，可供程式串接）
- 變動通知（設定條件自動推播）

## 資料來源

### 主要資料源

| 來源 | 網址 | 資料類型 | 更新頻率 |

### 監控資料類型

#### 央行政策利率
- 重貼現率
- 擔保放款融通利率
- 短期融通利率

#### 五大銀行新承做放款（台銀、合庫銀、土銀、華銀、一銀）
- 購屋貸款金額與利率
- 資本支出貸款金額與利率
- 週轉金貸款金額與利率
- 消費性貸款金額與利率

#### 指標利率
- 定儲利率指數（I指數）
- 各銀行一年期定存牌告利率
- 郵局定存利率

#### 存放款加權平均利率（全體本國銀行）
- 存款加權平均利率（更精確的無風險報酬基準）
- 放款加權平均利率（企業實際融資成本）
- 存放利差（銀行獲利能力指標）

### 資料源比較

```
五大銀行新承做放款 (5newloan)          存放款加權平均利率 (AVERAGEIR)
├── 僅五大公股銀行                     ├── 全體本國銀行
├── 僅「新承做」放款                   ├── 存量加權平均（反映整體市場）
├── 分類：購屋/資本支出/週轉金/消費性   ├── 同時包含存款與放款利率

互補關係：
• 5newloan → 觀察市場最新動向（新承做利率敏感度高）
• AVERAGEIR → 觀察整體市場水位（加權平均較穩定）
```

## 與股票投資分析的整合（GoodInfo.Analyzer）

本專案設計為可與 [Python-Actions.GoodInfo.Analyzer](../Python-Actions.GoodInfo.Analyzer) 整合，提供股票投資決策所需的宏觀經濟數據。

### 整合價值

> **限制：宏觀經濟連結不足** - 缺少利率/通膨與產業景氣的系統化連結，建議加入 CPI/利率時間序列與產業相關矩陣

### 利率對股票估值的影響

```
折現率決定因素：
├── 風險感知 → 風險越高 → 折現率越高 → P/E 越低
└── 通貨膨脹率（利率）→ 通膨越高 → 折現率越高 → P/E 越低
```

| 利率環境 | 對股票估值影響 | P/E 倍數傾向 |
|---------|--------------|-------------|
| 低利率 | 折現率低，未來獲利現值高 | 較高（15x-30x）|
| 高利率 | 折現率高，未來獲利現值低 | 較低（9x-19x）|

### 應用場景

1. **升息週期選股策略**
   - 利率上升 → 高殖利率股、金融股相對有利
   - 利率上升 → 高成長但不配息的股票承壓

2. **房貸利率與營建股**
   - 房貸利率↑ → 購屋需求↓ → 營建股營收壓力
   - 可作為營建股的**領先指標**

3. **融資利率與散戶行為**
   - 融資利率↑ → 融資成本增加 → 融資餘額可能下降
   - 結合 GoodInfo 的融資情緒分析

### 具體應用效益

#### 1. P/E 估值動態校準

```
目前：固定 P/E 倍數區間（9x-30x）

改善後：
P/E 合理倍數 = f(無風險利率, 風險溢酬)

當央行升息 1.875% → 2.0%：
├── 無風險報酬上升
├── 股票要求報酬率上升
└── 合理 P/E 應下修約 5-10%
```

#### 2. 殖利率投資決策

```python
# 目前：僅看股票殖利率
股票殖利率 = 5%  # 看起來不錯？

# 改善後：與無風險利率比較
股票殖利率 = 5%
定存利率 = 1.5%  # ← 從 TaiwanLoanInterest 取得
殖利率利差 = 3.5%  # 有足夠風險補償

# 若定存利率升至 3%
殖利率利差 = 2%  # 風險補償不足，需重新評估
```

#### 3. 產業前瞻指標

| 利率數據 | 可預測的產業影響 |
|---------|----------------|
| 房貸利率 ↑ | 營建股營收壓力（領先 3-6 個月）|
| 企業貸款利率 ↑ | 資本密集產業（如面板、半導體）擴廠意願降低 |
| 消費性貸款利率 ↑ | 耐久財消費（汽車、家電）需求下降 |

#### 4. 融資情緒修正

```
GoodInfo 現有：融資餘額、融資使用率
               ↓
TaiwanLoanInterest 補充：融資利率成本
               ↓
整合分析：
├── 融資利率 6% + 融資餘額高 → 散戶成本壓力大，潛在賣壓
└── 融資利率 6% + 融資餘額低 → 市場已去槓桿，底部訊號
```

### 量化效益範例

假設投資組合 1000 萬元：

```
情境：央行連續升息，但 GoodInfo 未調整估值模型

Without TaiwanLoanInterest:
├── 使用固定 P/E 15x 買進
├── 實際合理 P/E 應為 12x（因利率上升）
└── 潛在損失：20% = 200 萬元

With TaiwanLoanInterest:
├── 偵測到利率上升趨勢
├── 動態下修 P/E 至 12x
├── 避免在高估時買進
└── 保護資產價值
```

## 與現有平台的差異化

| 現有平台痛點 | 本系統解決方案 |
|------------|--------------|
| 商業導向，夾帶廣告推銷 | 純數據、無商業置入 |
| 資料更新不即時 | 自動化爬蟲即時更新 |
| 缺乏歷史原始數據下載 | 提供 API 與資料匯出 |
| 各平台資料格式不一 | 標準化資料結構 |
| 無客製化警報功能 | 利率變動推播通知 |
| 僅呈現當下，缺乏趨勢預測 | 結合總經指標分析 |
